# Prompt Context Documentation

This folder contains documentation files that provide context for AI assistants working with this codebase. These files are not part of the public documentation but contain useful technical details, troubleshooting guides, and implementation notes.

## Contents

- **STRIPE_INTEGRATION.md** - Technical details about Stripe Payment Links integration, webhook handling, and architecture decisions
- **TESTING_GUIDE.md** - Step-by-step testing workflows and edge cases
- **POSTGRESQL_SETUP.md** - Database configuration and setup details
- **DEBUGGING_EMAILS.md** - Email troubleshooting guide and common issues
- **FIX_EMAIL_AUTH.md** - Quick reference for email authentication problems
- **BREVO_CHECKLIST.md** - Brevo (email service) configuration checklist

These documents help provide context about:
- How the system works internally
- Common issues and their solutions
- Architecture decisions and trade-offs
- Integration details with third-party services
